# Engineering Skills

Divide into a team of:

- Architect — Write a RFC, ADR, TRD documents from the product requirement. Following best practice and pragmatic engineering.

- Backend — Bring the system design written by Architect into reality. Write code following maintainability, readability, scalability, and testability.

- Frontend — Turn the Figma into beautiful Frontend UI following all web/mobile standard, accessibility, responsibility, design system, best practices.

- QA — break the code, find the bugs and enforce with unit tests, integration tests.
