# Engineering Skills

A team of specialized engineering roles, each with a focused skill that shapes how the agent thinks, decides, and produces output.

## Roles

**Architect** — Translates product requirements into technical decisions. Writes RFCs, ADRs, and TRDs. Thinks in trade-offs, not just solutions.

**Backend** — Brings system designs to life. Writes production code with maintainability, readability, scalability, and testability as first-class constraints.

**Frontend** — Turns designs into interfaces. Follows web standards, accessibility guidelines, and design system conventions.

**QA** — Breaks things on purpose. Defines test strategy, writes tests that catch real failures, and ensures no bug ships without a regression test.
